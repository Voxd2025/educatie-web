# ABACUS Web – GitHub Pages pakket

Dit pakket is klaar om op GitHub Pages te zetten.

INHOUD
- index.html  -> de webapp
- .nojekyll   -> zorgt dat GitHub Pages het bestand direct publiceert

ZO ZET JE DIT ONLINE VIA GITHUB PAGES

1. Maak een nieuwe repository
Gebruik bij voorkeur een nieuwe public repository, bijvoorbeeld:
abacus-web

2. Upload de bestanden
Upload uit dit pakket:
- index.html
- .nojekyll

3. Zet GitHub Pages aan
Ga in GitHub naar:
Settings -> Pages

Kies daarna:
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

Klik op Save.

4. Wacht even
GitHub maakt dan een webadres aan. Dat duurt meestal 1 a 2 minuten.

Je site komt dan ongeveer op:
https://JOUW_NAAM.github.io/abacus-web/

5. Open op de iPad
Open de GitHub Pages-link in Safari.

6. Zet op beginscherm
In Safari:
- tik op het deel-icoon (vierkant met pijl omhoog)
- kies 'Zet op beginscherm'

BELANGRIJK
- Voor gratis GitHub Pages moet de repository meestal public zijn.
- Gebruik hiervoor liever niet je bestaande portfolio-repository, tenzij je daar bewust deze app in wilt plaatsen.

TIP
Wil je later een andere versie uploaden?
Vervang dan gewoon index.html in dezelfde repository.
