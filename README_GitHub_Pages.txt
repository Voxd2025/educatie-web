naar idee van ABACUS/DeBasis V3.0.1:  Beat The Calculator! - Eugen O.

Bestanden:
- index.html
- .nojekyll
- README_GitHub_Pages.txt

Upload naar GitHub:
1. Open repository educatie-web.
2. Vervang de bestaande bestanden door deze drie bestanden.
3. Commit changes.
4. Wacht tot GitHub Pages klaar is met bouwen.
5. Test eerst in Safari/Edge via de normale URL, niet via een oud beginscherm-icoon.

Inhoud versie 10:
- Alle moeilijkheidsgraden hebben +, -, × en ÷ actief en selecteerbaar.
- Oude localStorage-instellingen worden automatisch gemigreerd naar alle vier bewerkingen.
- Administrator-wachtwoordbeheer staat zichtbaar boven de analyse.
- Standaard administrator-wachtwoord: oefenen.
- Administrator-wachtwoord kan lokaal worden aangepast.
- Gebruikersnamen kunnen worden bewerkt/verwijderd.
- Analyse gebruikt berekende waarden per operatie.
- CSV-export bevat goed/fout/beantwoord.
- JSON backup/import aanwezig voor volledige data.

Controle:
- JavaScript-syntax gecontroleerd met node --check.
- HTML gecontroleerd op aanwezigheid van adminPasswordInput, saveAdminPasswordBtn en usersAdminBox.
